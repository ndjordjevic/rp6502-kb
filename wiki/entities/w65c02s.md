# W65C02S

**Summary**: The Western Design Center CMOS 65C02 — the actual CPU in the Picocomputer. Slot **U1** on [[rp6502-board]]. No substitutions allowed; NMOS 6502s are not supported.

---

## Role

- Sole general-purpose CPU. The RP2350 in [[rp6502-ria]] does **not** emulate or offload the 6502; it only provides peripherals, clocks, and the OS.
- Runs at variable **0.1 – 8.0 MHz** (PHI2 generated by [[rp6502-ria]]).
- PHI2 and RESB are **owned by the RIA** — see [[reset-model]].

## Memory map as seen by this CPU

See [[memory-map]] for the full picture.

| Range | Content |
| --- | --- |
| `$0000-$FEFF` | RAM (63.75 K) |
| `$FF00-$FFCF` | Unassigned (user chip-select expansion) |
| `$FFD0-$FFDF` | [[w65c22s]] VIA |
| `$FFE0-$FFFF` | [[rp6502-ria]] registers |

Note: zero page is **not reserved** — every program starts with a blank 6502.

## Hardware substitution

From [[hardware]]:
- Must be a genuine WDC **W65C02S** (part `W65C02S6TPG-14`).
- **No NMOS**: don't try a 6502 without the "C" in the part number.
- Old CMOS variants "may work" but there are no plans to support out-of-production parts.

## Open questions

- Exact speed ceiling with different glue-logic CMOS families (AC vs. HC) — the source only says "two of three must be AC for 8 MHz".
- Whether the W65C02S's WAI and STP instructions interact usefully with the RIA's `RIA_SPIN` polling mechanism (see [[rp6502-abi]]).
